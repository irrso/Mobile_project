## 메인 아이디어
- 매번 어떤 음식을 먹을지 고르기 힘든데 누가 정해줄 수 없을까 ❓
- 비선호 음식, 카테고리를 기반으로 음식과 음식점을 동시에 추천해주자 💡
- 크롤링과 네이버 지도 API를 활용해 안드로이드 앱 제작 📲
<br>
